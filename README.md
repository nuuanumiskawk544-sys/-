## 📜 幸存者守则
1. **故事线规划**：按照核心法则4条线进行（每10章一个循环），注意剧情节奏，紧张刺激的打脸过程可以详细展开细说，无关这四条线的地方尽快过度，感情线要体现情感深度，循序渐进，让文章整体节奏慢下来，保证一松一紧，松弛有度
2. **经典四合院打脸**： 集合了易中海、贾张氏、秦淮茹、许大茂等所有经典“禽兽”角色，进行酣畅淋漓的降维打击。
3. **空间物资流爽点**：主角拥有近乎无限的顶级物资，与时代的匮乏形成强烈对比，满足读者的“仓鼠症”和富足感。
4. **极致羞辱，精神打击**：主角的所有行为最终都获得了官方支持，从“个体户”升级为“国家顾问”，安全感十足。不仅在物质上碾压众禽，更在精神上让他们崩溃发疯（啃树皮、家庭内讧），解气程度拉满。
5. **断绝后路，永不原谅**：最终彻底断绝与众禽的任何关系，不搞圣母原谅戏码，符合当下主流读者期待。
## 👁️ 核心世界观 (The Core Lore)
主角携带“十亩灵田”穿越物资匮乏的1965年，利用空间内无限产出的顶级食材，一方面在物质上极尽奢华馋哭邻居，另一方面利用“钓鱼执法”和“精神折磨”将满院禽兽玩弄于股掌之间，主打一个“我吃肉你喝风，还要被我打肿脸”。
* **核心法则**：
    1.  **金手指升级线**：灵田种植 （动物只能养鸡）-> 养殖（全部动物都可以养） -> 农产品粗加工坊，同时开放鱼类养殖（只能养淡水鱼） -> 农产品精细加工坊（全部鱼类，包括海鱼）->初步的小型手工业加工坊->专业级精工工业加工坊->初步工业兵工厂残骸修复 -> 未来科技实验室。总之要一步一步升级，不能通过一次升级直接实现工业革命式的大跨步升级！
    2.  **打脸线**：怼贾张氏 ，整治棒梗，整治秦淮茹的同时将傻柱拉到同一阵线-> 气疯易中海（一大爷），整治二大爷，三大爷 -> 整治傻柱 -> 送许大茂去劳改 ，宗旨是要对四合院里的禽兽们反复打脸，但是不能全部打死，要来回拉扯！
    3.  **事业线**：一级钳工开始->转入后勤采购部门->升职部门主任->升职部门副部长->单独成立一个部门专职采购特殊农产品（空间产物）
    4.  **感情线**：可以加入同类年代文作品女主（电影《牧马人》中的女主角李秀芝）作为本书女主，感情循序渐进，缓慢推进，要通过打脸禽兽等方式体现保护女主的护妻感，（感情副线：娄晓娥的暧昧与商业合作）。
