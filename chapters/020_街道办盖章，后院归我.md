# 第20章：街道办盖章，后院归我

街道办的门敞着，王主任正低头看文件。

林东来敲了敲门框。

“王主任。”

王主任抬头，看见是他，脸上露出笑容：“东来啊，进来坐。怎么，有事儿？”

林东来走进屋，在办公桌对面的椅子上坐下。

“是有个事儿，想跟您汇报一下。”他说，“我老家有个远房表妹，叫苏云秀，家里遭了灾，父母都没了，一个人来京城投奔。现在没地方住，我想让她暂时住我那儿。”

王主任放下手里的钢笔：“表妹？以前没听你提过。”

“远房的，八竿子打不着那种。”林东来说，“但人已经来了，总不能让她流落街头。我后院那间空房，一直锁着，让她先住着，也算有个落脚地。”

王主任沉吟片刻：“这倒不是不行。但东来，我得提醒你，院里人多嘴杂，你一个大小伙子，让个姑娘家住后院，传出去不好听。”

“我知道。”林东来点头，“所以想请您给开个证明，就说街道办安排的临时安置。再给她办个临时户口，以后找工作也方便。”

“证明好开。”王主任拉开抽屉，拿出公章和信纸，“临时户口得去派出所办，我这儿只能开介绍信。不过……”

她顿了顿，看着林东来：“东来，你跟主任说实话，这姑娘真是你表妹？”

林东来面不改色：“真是。您要是不信，可以查她户口本——虽然她家那边遭了水灾，户口本泡烂了，但街道上应该有底子。”

这话说得滴水不漏。

王主任盯着他看了几秒，笑了：“行，我信你。不过东来，院里那些人，尤其是贾张氏，肯定得嚼舌根。你得有心理准备。”

“她们爱嚼就嚼。”林东来说，“我林东来行得正坐得直，不怕这个。”

“好，有骨气。”王主任拿起笔，在信纸上刷刷写了几行字，盖上公章，“这是介绍信，你拿着去派出所办临时户口。后院那间房，街道办同意暂时借给苏云秀同志居住，期限……先定三个月。”

“谢谢王主任。”林东来接过来。

“别急着谢。”王主任说，“东来，我听说你在厂里表现不错，后勤科的李科长还夸你采购有门路？”

林东来心里一动：“李科长抬举了，我就是运气好，认识几个朋友。”

“运气也是本事。”王主任意味深长地说，“好好干，以后有前途。对了，下个月街道要组织慰问军属，需要一批鸡蛋和猪肉，你要是能弄到，价格好说。”

“我试试。”林东来没把话说死。

“那就这么定了。”王主任站起来，“去吧，赶紧把事儿办了。”

林东来出了街道办，直奔派出所。

派出所的民警看了介绍信，又问了苏云秀的基本情况，没多刁难，就给办了临时户口——一张盖着红章的纸，上面写着苏云秀的名字、籍贯，还有暂住地址：南锣鼓巷95号后院。

办完这些，已经快中午了。

林东来没急着回院，先去副食店转了一圈。

副食店门口排着长队，都是等着买菜的。这个点儿，新鲜菜早就卖光了，只剩下些蔫了吧唧的土豆白菜。肉案子上空空如也，连点油星都看不见。

排队的人个个脸色发黄，眼巴巴盯着柜台。

“同志，还有鸡蛋吗？”一个老太太问。

售货员头也不抬：“没了，明儿早点来。”

“那肉呢？”

“肉？您想什么呢，这个月肉票早用完了。”

老太太叹了口气，拎着空篮子走了。

林东来站在队伍外面看着，心里有数了。

这年头，物资是真缺。

他转身离开，找了个没人的胡同，从空间里拿出两只宰好的鸡、二十个鸡蛋，用麻袋装好，拎着往回走。

刚进四合院大门，就听见中院传来贾张氏的骂声。

“小贱蹄子！让你扫个地都扫不干净，白吃我家粮食！”

接着是秦淮茹的声音：“妈，您别骂了，小当还小……”

“小什么小！五岁了还不会干活，养着有什么用！”贾张氏嗓门更大，“还有你，秦淮茹，这个月工资呢？怎么才交这么点？”

林东来脚步没停，径直往后院走。

经过中院时，看见贾张氏正叉着腰，指着秦淮茹骂。秦淮茹低着头，手里拉着小当。小当脸上挂着泪，吓得直往妈妈身后躲。

棒梗蹲在门口，眼睛盯着林东来手里的麻袋。

“林叔，您这拎的什么呀？”棒梗站起来，凑过来。

林东来没搭理他。

棒梗不死心，伸手就要摸麻袋：“让我看看呗……”

“滚。”林东来一个字。

棒梗手僵在半空，脸上挂不住：“我就看看怎么了？小气！”

贾张氏听见动静，转过头来，看见林东来，眼睛一亮：“哟，东来回来了？这拎的什么好东西，让大娘瞧瞧。”

说着就要过来。

林东来脚步加快，直接进了后院，反手把门闩上。

门外传来贾张氏的骂声：“什么东西！院里谁家有点好东西不拿出来分分？就你特殊！小气吧啦的，活该你爹妈死得早！”

林东来没理会。

他把麻袋放在地上，打开门。

苏云秀正在屋里擦桌子，听见动静，赶紧出来：“林大哥，您回来了。”

“嗯。”林东来把介绍信和临时户口递给她，“办好了。这是街道开的证明，这是临时户口，你收好。”

苏云秀接过来，看着纸上的红章，眼圈有点红：“谢谢林大哥……”

“别谢了。”林东来说，“后院那间房，钥匙在这儿。你自己收拾收拾，缺什么跟我说。”

他从兜里掏出一把钥匙。

苏云秀接过钥匙，手有点抖。

“还有这个。”林东来指了指地上的麻袋，“两只鸡，二十个鸡蛋，你先拿着。米面我屋里还有，等会儿给你拿点。”

苏云秀看着麻袋，愣住了：“这……这太贵重了……”

“让你拿着就拿着。”林东来说，“住我这儿，不能饿着。你会杀鸡吗？”

“会。”苏云秀点头，“我在家经常干。”

“那行，晚上炖一只。”林东来说，“另一只腌起来，留着慢慢吃。”

苏云秀蹲下身，打开麻袋。

两只鸡肥嘟嘟的，羽毛已经拔干净，内脏也掏空了。鸡蛋个个大，壳是淡粉色的，一看就是好蛋。

她抬头看林东来，眼神复杂：“林大哥，您对我这么好，我……我不知道怎么报答……”

“好好干活就行。”林东来说，“去收拾屋子吧，缺什么列个单子，我明天去买。”

苏云秀站起来，拎着麻袋往后院那间房走。

林东来回到自己屋，从空间里拿出二十斤白面、十斤大米，装在布袋里，给她送过去。

苏云秀正在扫地，屋里灰尘飞扬。

“先别扫了。”林东来说，“把这些放好。”

苏云秀接过布袋，一掂量，就知道分量不轻。

她没再说什么，把东西放在墙角，继续扫地。

林东来站在门口看了一会儿。

这姑娘干活确实利索，扫地、擦窗、铺床，动作又快又稳。虽然穿着旧衣服，但身段在那儿摆着，弯腰的时候，裤腿绷紧了，能看出小腿的线条。

“林大哥，您看这样行吗？”苏云秀铺好床，回头问。

“行。”林东来说，“缺被褥吗？”

“我有。”苏云秀从随身带的包袱里拿出一床薄被，“这个够用了。”

林东来看了一眼，那被子又薄又旧，棉花都结块了。

“明天我去买床新的。”他说。

“不用不用！”苏云秀赶紧摆手，“这个挺好的，真的。”

林东来没接话。

他转身回屋，从空间里拿出一床新棉被——系统仓库里什么都有，都是之前囤的。

抱着被子过来时，苏云秀正在擦窗户。

“这个给你。”林东来把被子放在床上。

苏云秀回头，看见那床崭新的棉被，愣住了。

被子是红底碎花的，棉花蓬松，一看就是好货。

“林大哥，这太……”

“让你用就用。”林东来打断她，“晚上冷，别冻着。”

苏云秀咬着嘴唇，眼睛又红了。

“别哭。”林东来说，“我这儿不兴这个。”

苏云秀赶紧擦擦眼睛：“我没哭……就是……就是……”

“就是什么？”林东来走近两步。

两人距离很近，能闻到对方身上的味道。

苏云秀身上有股淡淡的皂角香，混着汗味，不难闻。

林东来身上则是干净的肥皂味，还有一丝若有若无的烟草味——虽然他不抽烟，但空间里有囤的烟，偶尔拿出来闻闻。

“就是……觉得您太好了。”苏云秀小声说。

“我好？”林东来笑了，“院里那些人可不这么觉得。”

“他们是他们。”苏云秀抬起头，看着林东来，“我知道您是好人。”

林东来没说话。

他伸手，把苏云秀耳边的一缕碎发别到耳后。

手指碰到她的耳朵，有点凉。

苏云秀身子一僵，没躲。

“耳朵挺软。”林东来说。

苏云秀脸红了，低下头。

林东来收回手：“收拾完早点做饭，我饿了。”

“好。”苏云秀声音更小了。

林东来转身出了屋。

回到自己房间，他靠在椅子上，点了根烟——真点着了，抽了一口。

烟雾缭绕中，他眯起眼睛。

苏云秀这姑娘，比他想的还懂事。

知道感恩，知道分寸，关键是眼神干净，不像院里那些女人，满肚子算计。

留她在后院，既能帮忙干活，又能当个挡箭牌——以后他从空间拿东西出来，可以推说是“表妹带来的”。

一举两得。

正想着，系统面板跳出来：

【来源：贾张氏】

【怨气值+10】

【来源：棒梗】

【怨气值+5】

林东来嘴角微扬。

这才哪儿到哪儿。

等晚上炖鸡的香味飘出去，有他们受的。

他掐灭烟，起身去厨房。

得先把炉子生起来，等会儿苏云秀好做饭。

刚把炉子捅开，就听见前院传来吵嚷声。

是贾张氏，嗓门大得能掀翻房顶：

“易中海！你管不管！林东来那小子带个野女人回来，还关起门来不知道干什么呢！这要传出去，咱们院的名声还要不要了！”

接着是易中海的声音：“老嫂子，您别嚷嚷，这事儿得问清楚……”

“问什么问！我都看见了！那女的在他屋里待了一上午，现在又往后院搬东西！两只鸡！二十个鸡蛋！白面大米！他林东来哪来这么多好东西？肯定是投机倒把！”

林东来冷笑。

来了。

他放下火钳，不紧不慢地往外走。

刚到中院，就看见贾张氏拉着易中海，唾沫星子乱飞。

院里其他人也围过来了，二大爷刘海中、三大爷阎埠贵都在，还有几个看热闹的邻居。

秦淮茹站在自家门口，眼神复杂地看着后院方向。

“东来，你来得正好。”易中海看见他，摆出管事大爷的架子，“贾家嫂子反映，你带了个女同志回来，还往屋里搬了不少东西。这事儿，你得跟大家解释解释。”

林东来扫了一圈，目光落在贾张氏脸上。

“解释什么？”他问。

“解释那女的是谁！解释你那些东西哪来的！”贾张氏跳着脚，“我告诉你林东来，咱们院可是先进大院，不能藏污纳垢！你要是不说清楚，我就去街道办举报你！”

林东来笑了。

“去啊。”他说，“现在就去。街道办王主任刚给我开的证明，派出所刚办的临时户口，你去举报，看人家信你还是信我。”

贾张氏一愣：“什么证明？”

“苏云秀是我远房表妹，家里遭了灾，来投奔我。”林东来从兜里掏出介绍信的副本，“这是街道办开的证明，白纸黑字红章，要不要看看？”

他把纸抖开，举起来。

院里人都伸脖子看。

虽然离得远看不清字，但那红章看得清清楚楚——街道办的公章。

贾张氏脸色变了：“你……你哪来的表妹？以前怎么没听你说过？”

“我家的亲戚，还得跟你汇报？”林东来收起证明，“贾大妈，您管得也太宽了吧？”

“你！”贾张氏气得哆嗦，“那……那些东西呢？鸡、鸡蛋、白面大米，你哪来的？是不是投机倒把？”

林东来看着她，一字一句：

“我林东来行得正坐得直，每一分钱都是劳动所得。至于东西哪来的——关你屁事。”

院里一片安静。

贾张氏张着嘴，半天说不出话。

易中海咳嗽一声：“东来，话不能这么说。院里邻居关心你，也是为你好……”

“为我好？”林东来打断他，“一大爷，您要真为我好，就管管贾大妈，别让她整天盯着别人家锅里。有这功夫，不如想想怎么让自家孩子吃饱饭——棒梗都十岁了，还整天在院里偷鸡摸狗，传出去，咱们院的名声才真完了。”

这话戳到痛处了。

贾张氏嗷一嗓子：“你说谁偷鸡摸狗！”

“我说谁，谁心里清楚。”林东来转身，“没事儿我回去了，还得吃饭呢。”

他走了两步，又回头，看着易中海：

“对了，一大爷，后院那间房，街道办已经批给苏云秀同志暂住了。以后那就是她的屋子，谁要敢随便进去，别怪我不客气。”

说完，径直回了后院。

门闩落下。

中院，贾张氏一屁股坐在地上，拍着大腿哭嚎：

“没天理啊！小辈欺负老人啊！易中海你管不管啊！”

易中海脸色铁青，转身就走。

刘海中摇摇头，背着手回家了。

阎埠贵推了推眼镜，小声嘀咕：“两只鸡……二十个鸡蛋……这林东来，是真有门路啊……”

秦淮茹拉着小当回屋，关上门。

屋里，棒梗咬着牙：“妈，我想吃鸡……”

“吃吃吃，就知道吃！”贾张氏从地上爬起来，一巴掌拍在棒梗头上，“有本事你去后院抢啊！”

棒梗捂着头，眼睛盯着后院方向，满是怨恨。

后院。

林东来靠在门上，听着外面的动静。

系统面板又跳了：

【检测到怨气值】

【来源：贾张氏】

【怨气值+20】

【来源：棒梗】

【怨气值+10】

【来源：易中海】

【怨气值+5】

【当前怨气值：272/500】

快了。

再攒攒，就能升级了。

他转身，看见苏云秀站在厨房门口，手里拿着菜刀，脸色发白。

“林大哥，是不是……是不是我给您惹麻烦了？”她小声问。

“不是。”林东来说，“是她们自己找不痛快。你做饭，我去烧水。”

苏云秀点点头，进了厨房。

林东来把炉子捅旺，坐上水壶。

不一会儿，厨房里传来剁鸡的声音。

接着，是下锅的刺啦声。

香味，慢慢飘出来了。

灵泉养的鸡，肉质本就鲜美，再加上苏云秀手艺不错，葱姜蒜爆香，鸡肉下锅翻炒，那味道……

穿透窗户，飘到中院。

贾家屋里，棒梗趴在窗户上，鼻子使劲吸。

“奶奶，好香啊……”

贾张氏咽了口唾沫，骂骂咧咧：“香什么香！肯定是偷的！投机倒把！不得好死！”

但骂归骂，她肚子也咕咕叫。

中午吃的窝窝头咸菜，早就消化完了。

这会儿闻着炖鸡的香味，嘴里直冒酸水。

小当哭了：“妈，我饿……”

秦淮茹抱着女儿，眼圈红了。

她看着后院方向，心里五味杂陈。

林东来……

他到底哪来这么多好东西？

后院。

鸡炖好了。

苏云秀盛了一大碗，端到林东来屋里。

金黄的鸡汤，上面飘着油花，鸡肉炖得烂糊，筷子一夹就脱骨。

“林大哥，您尝尝咸淡。”苏云秀说。

林东来夹了一块，放进嘴里。

肉质鲜嫩，汤汁醇厚，灵泉的甘甜混着鸡肉的香，在舌尖炸开。

“不错。”他说，“你也吃。”

苏云秀盛了一小碗，坐在旁边吃。

她吃得很慢，每一口都细细品味。

林东来看着她，忽然问：“怕吗？”

苏云秀抬头：“怕什么？”

“院里那些人。”林东来说，“她们不会消停的。”

苏云秀放下碗，认真地说：“林大哥，我不怕。您对我好，我知道。她们要是敢欺负您，我……我跟她们拼了。”

林东来笑了。

“用不着你拼。”他说，“好好吃饭，好好干活。其他的，有我。”

苏云秀看着他，眼睛亮亮的。

“嗯。”

两人吃完饭，天已经黑了。

苏云秀收拾碗筷，林东来点了根烟，站在门口。

中院，贾家屋里传来贾张氏的骂声，还有棒梗的哭闹。

隐约能听见“我要吃鸡”、“凭什么他们能吃”之类的话。

林东来吐了口烟圈。

这才第一天。

往后，日子还长着呢。

他转身回屋，关上门。

系统面板又跳了：

【检测到怨气值】

【来源：贾张氏】

【怨气值+15】

【来源：棒梗】

【怨气值+10】

【当前怨气值：297/500】

快了。

再等几天，就能解锁新功能了。

林东来躺在床上，闭上眼睛。

明天，还得去厂里一趟。

李科长那边，得给个准信。

还有街道办王主任要的鸡蛋猪肉，也得准备。

事情一件接一件。

但，有意思。

比前世当社畜，有意思多了。

他翻了个身，睡了。

后院另一间屋里，苏云秀躺在崭新的棉被里，闻着被子上阳光的味道，怎么也睡不着。

今天发生的一切，像做梦一样。

从差点被纠察队抓，到遇见林大哥，再到住进这间屋子，有饭吃，有床睡。
